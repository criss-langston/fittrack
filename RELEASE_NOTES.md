# Release Notes

## Latest Update

### Nutrition moved to macro-only logging

FitTrack no longer uses meal logging or food logging.

Nutrition is now tracked with direct daily macro entries:

- calories
- protein
- carbs
- fat

### Why this changed

This update makes nutrition tracking:

- faster
- simpler
- more mobile-friendly
- easier to use every day

### What was removed

- meal logging
- food logging
- food search
- custom food entries
- meal-based nutrition flow

### What was added

- direct macro logging
- editable daily calorie/macro targets
- daily macro entry history
- weekly averages
- macro progress bars

### Data migration note

Old meal and food log data does not carry into the new macro-only nutrition system.

### App direction

FitTrack remains:

- local-first
- device-local
- offline-friendly
- mobile-focused
- not backend-driven
